# canalyzer
A coins analyzer

# Installation

## First install from github
```
cd ~
git clone git@github.com:badele/canalyzer.git
cd canalyzer
python3 -m venv .
source bin/activate.fish
pip install -r requirements.txt

cd ~/canalyzer/canalyzer
git clone git@github.com:badele/canalyzer-cache.git
```
